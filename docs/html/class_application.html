Részletes leírás

Alkalmazás osztály. A main() példányosítja és indítja el.

A konstruktor elvégzi az alapvető példányosításokat és a signalok összekapcsolását. Tartalmazza a kommunikációs obejektumokat, adatparsolást, biztosítja az objektumok közötti kommunikációt.
